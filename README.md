# Artefatos do Projeto

## Análise do Impacto da Estruturação de Prompts na Qualidade de Respostas de Modelos de IA Generativa Aplicados a Problemas de Computação

Este repositório reúne os artefatos, dados brutos e resultados experimentais produzidos durante o desenvolvimento do projeto da disciplina de Estágio Obrigatório.

O objetivo do estudo é analisar como diferentes estratégias de estruturação de prompts influenciam a qualidade das respostas geradas por modelos de Inteligência Artificial Generativa aplicados a problemas da área de Computação.

## Estrutura do Repositório

### Prompts_projeto

Contém os textos integrais dos prompts utilizados nos experimentos.

### Resultados_brutos

Contém as respostas originais produzidas pelos modelos avaliados durante a execução dos experimentos.

Estrutura:

* ChatGPT
* Gemini
* Claude

Cada subpasta contém os arquivos de resposta correspondentes aos cenários experimentais analisados.

### Avaliacoes

Contém os arquivos de avaliação e análise das respostas obtidas pelos modelos.

### matriz_resultados_consolidados.xlsx

Planilha contendo a consolidação dos resultados utilizados nas análises e comparações apresentadas no relatório.

### gerar_graficos.py

Script utilizado para geração dos gráficos empregados na apresentação dos resultados.

## Observação

Os materiais disponibilizados neste repositório têm finalidade exclusivamente acadêmica e são fornecidos para fins de transparência, auditoria e reprodutibilidade dos resultados apresentados no relatório do projeto.
